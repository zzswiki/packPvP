
# 🗡️ packPvP

Minecraft PvP resourcepack.


## 🪛 Installation

Download latest release and unzip it. Put unzip files to minecraft resourcepacks directory

```
Windows
%appdata%\.minecraft\resourcepacks
```

```
Linux
~/.minecraft/resourcepacks
```
